Consignas del Trabajo Practico

  • (15 pts.) El sitio web debe posser al menos 4 páginas HTML o 4 secciones si se trata de un sitio One Page
  • (15 pts.) El sitio web debe incluir un formulario de contacto, con al menos 5 campos que incluya (un checkbox o radiobutton), un select u una imagen, con validacion realizada mediante JavaScript para que los campos sean obligatorios
  • (15 pts.) Tener al menos una página totalmente responsive con minimo 3 puntos de corte con media querys para 3 tamaños de dispositivos (PC escritorio, Tablet, Celular). Para el diseño debe usarse CSS
  • (10 pts.) Utilizar al menos una animación, transformación o una transición
  • (20 pts.) Poseer una estructura HTML maquetada con Flexbox y/o Grid
  • (10 pts.) El sitio debe hacer una llamada GET a una API pública o a API de pruebas generada por el grupo (GET de un JSON).
  • (10 pts.) Se considerará la presentación general del proyecto, la legibilidad del mismo, la navegacíon sin llegar a puntos de no retorno, la optimización de imágenes para la web y el uso de favicon.
  • (Obligatorio!) El trabajo práctico deberá subirse a un servidor online y compartirse mediante un repositorio de Git
  • (Obligatorio!) La página deberá subirse a un servidor on-line para poder ser navegada por el Docente. Ejemplo: Netlify o similar.
  • (Obligatorio!) El sitio web debe estar estructurado utilizando etiquetas semánticas HTML correctamente Debe pasar la validación de https://validator.w3.org/ sin errores.
  • (5 pts.) Utilizar iframes y/o íconos de FontAwesome y/o fuentes locales o bien de Google Fonts. Es optativo incluir algún elemento de Bootsrap.
